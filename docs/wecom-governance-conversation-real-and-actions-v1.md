# 会话详情页真实化与治理入口表单 v1

更新时间：2026-03-30

## 1. 本轮新增

治理台详情页已进一步推进：
- real 模式下优先调用 `/api/v1/wecom/conversations/:conversationId`
- 治理入口区已从占位按钮推进为可交互表单骨架
- 当前最小支持动作：
  - confirm
  - reassign
  - unconfirm

---

## 2. 当前取舍

为了保持最小实现，这一版还没有：
- 完整患者搜索
- 完整治理完成后的自动刷新链
- promote-binding 表单

但已经具备：
- 真实详情优先读取
- 真实治理动作最小提交能力

---

## 3. 一句话结论

详情页已经开始从“页面骨架”推进到“真实详情 + 可提交治理动作”的过渡阶段。
