# 治理台会话详情页骨架 v1

更新时间：2026-03-30

## 1. 本轮新增

前端治理台已从首页列表页进一步推进到最小会话详情页骨架。

已具备：
- 首页 `查看会话` / `去治理` 可进入详情页
- 详情页带 `conversationId`
- 详情页包含：
  - 基本会话信息
  - 当前问题状态
  - 最近治理动作
  - 治理入口区占位

---

## 2. 当前实现方式

为了保持最小实现，这一版没有额外引入路由库，而是：
- 使用 URL query + 前端 view state 进行页面切换

当前关键参数：
- `view=conversation-detail`
- `conversationId=...`

---

## 3. 一句话结论

治理台已经从“首页可看”推进到“可以进入最小详情页骨架”，页面链路开始具备真实后台的列表 -> 详情结构。
