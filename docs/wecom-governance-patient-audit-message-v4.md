# 详情页继续增强：会话-候选关联 / audit 总结条 / 可判断消息筛选 v4

更新时间：2026-03-30

## 本轮新增

### 1. patient 候选与当前会话更强关联
- 候选区尝试补拉 `/api/v1/patients/:id` 详情
- 候选卡片可展示：
  - 最近会话数量 / 最近会话名
  - 最近洞察摘要
- 当前映射卡新增会话侧上下文：
  - 当前 conversationId
  - 当前主客户（占位表达）
  - 当前 matchedBy

### 2. audit 动作结果总结条
- 最新 audit 区新增总结条
- 当前支持直接表达：
  - 已从 A 改到 B
  - 已提升为 binding
  - 已撤销人工确认
  - 已确认到某个 patient

### 3. 只看可判断消息
- 消息区新增“全部消息 / 只看可判断”切换
- 过滤规则：
  - 去掉 system 类消息
  - 去掉空内容消息
  - 去掉 sender 看起来是 system 的消息

## 一句话结论

详情页已经进一步从“能操作”往“能快速判断”推进：候选和会话更有关系，audit 更直接告诉你结果，消息更接近治理判断视角。
