# 详情页继续增强：patient 选择 / 审计面板 / 消息展示 v2

更新时间：2026-03-30

## 本轮新增

### 1. patient 选择能力
- 详情页治理表单接入 `/api/v1/patients`
- 支持按 patientId / 姓名 / 手机做前端筛选
- 支持点击候选项回填 `patientId` 或 `toPatientId`
- 展示已选 patient 卡片

### 2. 审计结果面板增强
- 增加“最新审计结果”高亮区
- 审计列表首条高亮
- 增加 `toPatientId` 展示

### 3. 消息展示稳定性增强
- 增加时间排序切换（最新在前 / 最早在前）
- 长消息支持展开 / 收起
- sender / type / time 采用更稳定的字段兜底

## 一句话结论

详情页已经从“可提交治理动作”继续推进到“可辅助判断、可回填 patient、可确认审计结果、可稳定浏览消息”的更正式治理后台形态。
