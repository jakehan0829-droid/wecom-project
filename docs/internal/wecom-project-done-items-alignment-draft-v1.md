### 4.1 与“群管理机器人 / 群客服主线”相关

目前已覆盖得较少，主要还是基础层和承接层：
- 有企微接入基础

但从“真正群内互动、成员信息提炼、成员档案沉淀”这个角度看，当前更多还是：

- 真实试跑与标准样本已形成

但从“真实患者互动、问答、患者信息提炼、患者档案沉淀”这个角度看，当前更多完成的是：
- 工作台承接能力

### 4.3 与“互动沉淀档案主线”相关
