# 企微项目下一阶段执行清单 v1

## 1. 当前阶段结论

当前企微项目已完成第一轮 MVP 主链落地，已真实打通以下链路：

1. 消息入库闭环
2. 聊天分析闭环
3. 业务反哺闭环
4. 业务动作联动闭环
5. 真实企微格式接入验证
6. customerId -> patientId 自动映射第一轮验证

当前项目已从“设计/草案阶段”进入“下一阶段正式化推进”。

---

## 2. 下一阶段总目标

下一阶段不再以“证明能跑”为核心，而以“真实接入、稳定识别、自动推进、可运营”作为核心目标。

本阶段聚焦四类结果：

1. 真实企微消息正式接入
2. 客户识别与映射体系稳定化
3. 业务动作自动化增强
4. 运营与管理视图准备

## 5. 阶段完成判定

满足以下条件时，可认为“下一阶段第一轮完成”：

1. 真实企微消息可通过正式接收入口进入系统
2. 系统可稳定识别/映射大部分 customer -> patient 关系
3. 业务动作生成具备优先级与差异化逻辑
4. 动作执行结果可以回流
5. 管理视图所需核心数据口径明确
