# 用户操作手册

## 系统概述

企业微信糖尿病慢病管理系统是一款基于企业微信的智能医疗助手系统，帮助医生和医疗团队通过企业微信与患者互动，自动分析患者信息，完善患者档案，并提供个性化的健康管理建议。

## 用户角色

### 1. 医生/医疗团队
- 查看患者列表和详情
- 查看患者与AI助手的对话记录
- 审阅AI分析建议
- 手动完善患者档案
- 查看健康数据趋势

### 2. 患者
- 通过企业微信与AI助手交流
- 报告健康状态和问题
- 接收健康建议和提醒
- 查看个人健康档案

### 3. 管理员
- 管理系统配置
- 管理用户权限
- 查看系统运行状态
- 配置AI模型参数

## 快速入门

### 首次使用

#### 医生端
1. **登录系统**
   - 访问系统URL (如: http://localhost:5173)
   - 输入用户名和密码登录

2. **查看患者列表**
   - 登录后进入医生工作台
   - 左侧面板显示患者列表
   - 点击患者可查看详情

3. **查看患者档案**
   - 选择患者后，查看右侧档案面板
   - 查看基本信息、健康状况、历史记录

4. **查看对话记录**
   - 点击"消息"标签页
   - 查看患者与AI助手的完整对话
   - AI分析结果会显示在对话旁边

#### 患者端
1. **添加企业微信**
   - 扫描医生提供的二维码
   - 添加"糖尿病管理助手"为企业微信联系人

2. **开始对话**
   - 发送消息给助手，如"你好"或"最近血糖有点高"
   - AI助手会自动回复并提供建议

3. **报告健康数据**
   - 发送血糖值、血压值等数据
   - 系统会自动记录并分析

## 核心功能详解

### 1. 患者管理

#### 查看患者列表
- 列表显示患者姓名、性别、风险等级、管理状态
- 支持按姓名、风险等级、管理状态筛选
- 支持搜索功能

#### 患者详情页面
包含以下信息:
- **基本信息**: 姓名、性别、年龄、联系方式
- **健康档案**: 疾病类型、风险等级、核心问题
- **企业微信绑定**: 绑定状态、最近互动时间
- **最近会话**: 最近5次对话摘要
- **AI建议**: 最新的AI分析建议
- **待办事项**: 需要医生处理的事项

#### 患者档案编辑
1. 点击患者详情页的"编辑档案"按钮
2. 可编辑字段包括:
   - 基本信息 (JSON格式)
   - 个人偏好
   - 核心问题描述
   - 沟通要点
   - 随访重点
   - 近期问题总结
3. 保存后，变更会记录在变更日志中

### 2. 消息与对话

#### 查看对话记录
- 按时间顺序显示所有消息
- AI回复会标记为"AI助手"
- 患者消息标记为"患者"
- 系统消息标记为"系统"

#### AI分析结果
每条消息下方可能显示AI分析结果:
- **用户问题**: AI理解的患者核心问题
- **用户状态**: 患者的情绪和状态
- **新需求**: 识别出的新需求
- **关注点**: 需要关注的健康问题
- **风险点**: 潜在的健康风险

#### 消息搜索
- 支持按关键词搜索消息内容
- 支持按时间段筛选
- 支持按发送者筛选

### 3. 档案管理

#### 成员档案
- 每个企业微信成员对应一个档案
- 包含基本信息、偏好、问题总结等
- 随着互动自动更新

#### 患者档案
- 患者专属的健康档案
- 与成员档案关联
- 包含更详细的医疗信息

#### 档案搜索
- 支持按姓名、关键词搜索
- 支持按档案类型筛选
- 支持按更新时间排序

#### 批量操作
- 支持批量更新档案字段
- 支持导出档案数据
- 支持批量添加备注

### 4. AI分析功能

#### 自动消息分析
系统会自动分析每条患者消息:
1. **互动理解**: 理解用户意图和状态
2. **信息提炼**: 提取关键健康信息
3. **档案更新**: 自动建议档案更新内容
4. **风险评估**: 识别健康风险等级

#### 档案完善建议
AI会定期分析档案并提出完善建议:
- 补充缺失信息
- 更新过时信息
- 修正矛盾信息
- 添加新的观察点

#### 业务路由决策
根据消息内容自动决定处理方式:
- **群客服模式**: 适用于一般咨询
- **医疗助手模式**: 适用于健康问题咨询
- **人工转接**: 复杂问题转人工处理

### 5. 健康数据管理

#### 血糖记录
- 记录血糖测量值
- 支持不同测量类型 (空腹、餐后、随机)
- 显示趋势图表
- 设置正常范围提醒

#### 血压记录
- 记录收缩压和舒张压
- 记录测量时间和状态
- 显示血压趋势
- 高血压提醒

#### 体重记录
- 记录体重变化
- 计算BMI指数
- 显示体重趋势图表

#### 数据图表
- 血糖变化趋势图
- 血压波动图
- 体重变化曲线
- 综合健康评分

## 操作流程示例

### 示例1: 新患者接入流程

1. **患者扫码添加企业微信**
   ```
   患者 → 扫描二维码 → 添加"糖尿病管理助手"
   ```

2. **患者发送首条消息**
   ```
   患者 → "你好，我是新患者" → AI助手
   ```

3. **AI自动回复并开始建档**
   ```
   AI助手 → "欢迎！请告诉我您的姓名和基本情况"
   ```

4. **患者提供基本信息**
   ```
   患者 → "我叫张三，有2型糖尿病3年了"
   ```

5. **系统自动创建档案**
   ```
   系统 → 创建患者档案 → 记录基本信息
   ```

6. **医生查看新患者**
   ```
   医生 → 登录系统 → 查看新患者列表 → 查看张三档案
   ```

### 示例2: 日常健康咨询流程

1. **患者报告问题**
   ```
   患者 → "最近早上空腹血糖都在8以上，怎么办？"
   ```

2. **AI分析并回复**
   ```
   AI助手 → 
   分析: 空腹血糖偏高
   建议: 1.调整晚餐时间和内容 2.增加晨间运动
   提问: 最近饮食和运动有什么变化？
   ```

3. **系统更新档案**
   ```
   系统 → 更新档案 → 添加"近期问题: 空腹血糖偏高"
   ```

4. **医生审阅建议**
   ```
   医生 → 查看对话记录 → 确认AI建议 → 添加补充建议
   ```

### 示例3: 档案完善流程

1. **AI分析档案缺口**
   ```
   AI → 分析档案 → 发现缺少"用药历史"
   ```

2. **生成完善建议**
   ```
   AI → 建议: 请补充患者的用药历史和过敏史
   ```

3. **医生执行完善**
   ```
   医生 → 编辑档案 → 添加用药信息 → 保存
   ```

4. **系统记录变更**
   ```
   系统 → 记录变更日志 → 标记为"医生手动完善"
   ```

## 常见问题解答

### Q1: 患者如何开始使用？
A: 患者只需扫描医生提供的企业微信二维码，添加助手为好友，即可开始对话。

### Q2: AI助手的回复准确吗？
A: AI助手基于专业医疗知识库，但所有重要建议都会提示医生审阅。医生可以随时修改或补充AI建议。

### Q3: 如何确保患者隐私？
A: 系统采用企业级加密，所有数据存储在安全的服务器上，符合医疗数据保护标准。

### Q4: 可以自定义AI的回复风格吗？
A: 可以。管理员可以在系统设置中配置AI的回复语气、详细程度和专业术语使用程度。

### Q5: 系统支持哪些健康数据类型？
A: 目前支持血糖、血压、体重等基础数据，未来会扩展支持更多数据类型。

### Q6: 如何导出患者数据？
A: 在患者详情页点击"导出数据"，可以选择导出格式 (CSV, PDF, JSON) 和数据范围。

### Q7: 系统故障如何处理？
A: 系统有自动监控和告警功能。如遇问题，请联系技术支持或查看系统状态页面。
